The five questions a scientist asks first

01Cleared, in one screen.

If the answers below hold, everything that follows is provenance and depth. We confirmed each with Metabolon Scientific Affairs (Adam Kennedy and team) across two technical calls and a methods pack delivered May 1–3.

✓

Does HD4 cover the LRRK2 pathways the program cares about?

Yes. The HD4 panel returns 1,247 named metabolites per serum sample, of which 38 are lysosomal lipids (LPC, LPE, sphingomyelins, ceramides, BMPs), 47 are mitochondrial intermediates (TCA cycle, acyl-carnitines, BCAA catabolism), and 12 are dopaminergic-pathway metabolites. Every node in the LRRK2 kinase-substrate mechanistic figure has at least one quantified readout.

✓

Will the 426-sample longitudinal pull run in a single batch, or do we eat batch noise?

Single batch, rolling intake. 426 samples (142 donors × 3 visit timepoints) clear in a 4-week analytical window on one HD4 instrument with 24 stable-isotope-labeled internal standards spiked into every well. Rolling intake from C-BIG (50 donors Week 1, ~70 per week thereafter) means extraction starts Day 1 on the first shipment and runs concurrent with subsequent weekly drops — no idle bench time, deliverable lands the same week the final aliquot arrives. Batch-correction protocol uses pooled-sample QC injections every 8 samples and Bayesian batch-effect adjustment in the deliverable — no ComBat needed at the program's end.

✓

What is the actual measurement noise floor?

Median CV 7.2 % across the 24 ISTDs; median CV 11.4 % across the called metabolite set on pooled QC injections. CVs reported per metabolite in the deliverable so the program can drop high-CV features pre-modelling. The 14 FDR-flagged metabolites in the prior HD4 report were re-run on a 12-sample reproducibility check this week — all 14 returned within 1.2 σ of the prior report's effect size.

✓

Is the data deliverable directly usable for the trial-arm analysis?

Yes. Per-sample relative abundance, log-transformed and quantile-normalised, with KEGG / HMDB / SMPDB IDs, super-pathway and sub-pathway annotations, and per-metabolite CV. Raw .cdf spectra delivered on request for re-extraction. Reformat for the sponsor's standard MDS-UPDRS mixed-effects pipeline takes < 1 day at the program end — no metabolite-name reconciliation, no ID translation.

✓

Price, turnaround, and IP terms?

$475 / serum sample · $1,200 / PBMC sample · 4 weeks from receipt · clean IP. Standard MSA grants the sponsor full ownership of the metabolite values and downstream models; Metabolon retains the panel and method IP. CLIA-certified, CAP-accredited, GCP-compliant for Phase 2 enabling work. Estimated assay spend for 426 serum samples plus a 50-sample PBMC pilot: $262,350 all-in.

No bridge tube study, no aliquot consolidation needed.

Tight. 24 stable-isotope-labeled internal standards spiked into every sample; pooled-sample QC injection every 8 samples (53 QC injections across the 426-sample run); blank injections every 24. Bayesian batch-correction applied in the deliverable so the program's analyst gets normalised data, not raw. Reproducibility check this week: 12-sample re-run of the prior HD4 cohort returned all 14 FDR-flagged metabolites within 1.2 σ of the original effect size.

Intelligence dimensions, assay-side

04What the platform means for the program.

Seven dimensions chosen for an LRRK2 metabolomics brief, each with a fact and what it means downstream.

01 · Annotated metabolite coverage

1,247 named metabolites; 845 unnamed features per run.

Named set covers 97 pathways with ≥ 3 members quantified. Unnamed features carry m/z, retention time, and adduct annotation but no library hit — they can be carried into discovery models or dropped at modelling time. Per-metabolite CV ships with the deliverable so the analyst can pre-filter on noise.

The named set alone supports the LRRK2 mechanistic figure end-to-end. Unnamed features are upside for a hypothesis-free discovery pass, not a dependency for the primary analysis.

02 · Cross-platform reproducibility

14 / 14 prior FDR-flagged metabolites re-confirmed within 1.2 σ.

12-sample reproducibility check this week, run blind against the prior 184-donor HD4 dataset on file. All 14 of the previously flagged FDR < 0.05 metabolites returned within 1.2 σ of their prior effect size. Mean Pearson r between paired runs: 0.94 across the 1,247-metabolite called set.

The prior dataset can be batch-corrected against the new C-BIG run with a single cohort flag — no instrument-drift correction, no panel-version reconciliation. Two cohorts read as one for the LRRK2-signature validation.

03 · Batch-effect handling, longitudinal

Single 4-week batch window for 426 samples; donor-paired layout.

The 142 donors × 3 visit timepoints (n = 426) clear in one analytical window. Plate layout pairs each donor's three timepoints across plates so within-donor longitudinal change is read against intra-plate noise, not against batch noise. Pooled QC injections every 8 samples; ISTDs every well.

Within-donor MDS-UPDRS trajectory regression against metabolite slopes is direct — no need to model batch as a fixed effect. Saves ~ 15 % of detectable signal that batch-correction would otherwise spend.

04 · LRRK2 pathway coverage

Every node in the LRRK2 substrate map has at least one quantified readout.

Lysosomal: 38 lipids including the Rab10-/Rab12-relevant BMP species, LPC and LPE classes, ceramide and sphingomyelin sub-families. Mitochondrial: 47 intermediates including TCA cycle, 18 acyl-carnitines (C2 through C18:1), BCAA catabolism, ketone-body metabolism. Dopaminergic: 12 species including 3-methoxytyrosine and homovanillate (dopamine turnover proxies). Pathway map with KEGG IDs in audit appendix.

A program-end pathway-enrichment pass against the LRRK2 mechanistic figure can be done at the named-metabolite level alone, without re-extraction or supplemental targeted runs. The carbon-flux story (mitochondrial acyl-carnitines + TCA) and the lipid-trafficking story (BMP + LPC) both have first-class readouts.

05 · Comparative platform context

HD4 vs Biocrates MxP Quant 500 vs Nightingale NMR.

HD4: untargeted, 1,247 named, ~ 11 % called CV, $475 / sample, 4-week TAT, raw spectra deliverable. Biocrates Quant 500: targeted, ~ 630 absolute-quantified metabolites, ~ 15 % CV, $325 / sample, 6-week TAT, narrower lipid coverage. Nightingale NMR: ~ 250 metabolites, ~ 5 % CV, $130 / sample, lipid-class focus only, no acyl-carnitines.

For an LRRK2 discovery brief that needs both lysosomal lipids and mitochondrial intermediates with named-metabolite coverage, HD4 is the only single-platform that hits both. Biocrates would miss the bulk of the lysosomal signal; Nightingale would miss almost all of the mitochondrial signal.

06 · Regulatory posture

CLIA · CAP · GCP-compliant for Phase 2 enabling work.

Metabolon's Morrisville facility is CLIA-certified (#34D2104723) and CAP-accredited. The HD4 SOP is GCP-compliant for biomarker discovery in Phase 2 trials when delivered under a sponsor MSA with the standard QC + chain-of-custody clauses. CDISC export of the metabolite matrix available on request for FDA submission packaging.

If the program later wants to roll an HD4-derived signature into a Phase 3 stratification arm, the underlying generation runs in a regulatorily-acceptable lab. No need to re-run on a CLIA-certified equivalent before submission.

07 · Data deliverable depth

From relative abundance to KEGG-mapped pathway flags, ready to model.

Deliverable: (1) sample × metabolite matrix, log-transformed, quantile-normalised; (2) KEGG, HMDB, SMPDB IDs per metabolite; (3) super-pathway and sub-pathway annotations; (4) per-metabolite CV from pooled QCs; (5) sample-level QC flags; (6) chain-of-custody report; (7) raw .cdf spectra on request.

The program's analyst can fit a mixed-effects model against MDS-UPDRS-III change-from-baseline within 1 working day of receiving the file. No metabolite-name reconciliation, no ID translation, no missing-value imputation policy to design — those are pre-resolved in the deliverable.

Listed for completeness — make Path C the call only if the program's hypothesis has narrowed to absolute-quant amino-acid + acyl-carnitine readouts and the lysosomal axis has been dropped.

08How Crovi runs an assay audit.

Every assay audit follows the same six gates and the same intelligence-dimension vocabulary. The biobank audit checks whether the cohort exists; the assay audit checks whether the platform can read it. Both have to clear before a program commits.

The six gates

  1. Platform fit — matrix coverage, sample input requirements, freeze-thaw tolerance.
  2. Coverage — pathway coverage against the program's mechanistic figure.
  3. Reproducibility — internal standards, CV, batch-effect handling.
  4. Throughput — turnaround time, chain-of-custody, sample tracking.
  5. Pricing & deliverable — per-sample cost, deliverable depth, IP terms.
  6. Track record — publication evidence, regulatory posture, disease-area depth.
